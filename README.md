# wilders.csv

The goal of this challenge is to make you manipulate that list, using commands you've been shown.

Create a php_france_2019.csv file containing only the number of wilders who have learnt **PHP on a campus located in France, **in 2019.

Create a javascript_biarritz_toulouse.csv file containing only wilders who have learnt JavaScript on the campuses of Toulouse and Biarritz. Hint you might have to create "intermediate" files and concatenate them!

Combining the history and tail commands (by adjusting the number of lines), produce a history.txt file containing the commands you had to run to get to the previous results.
